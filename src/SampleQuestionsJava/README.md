## How to Compile and Run:

```bash
# Compile a specific question
javac SampleQuestionsJava/Question1.java

# Run the compiled class
java SampleQuestionsJava.Question1
```

## Package Structure:
All Java files are in the `SampleQuestionsJava` package and located in the `src/SampleQuestionsJava/` directory.

## Notes:
- Each question is implemented as a separate Java class
- Input/output follows the same pattern as the original C programs
- Proper error handling has been added where appropriate
- Modern Java features like StringBuilder, ArrayList, and HashMap are used where beneficial
